What is JS variables

JavaScript variables are containers used to store data values like numbers, text, or objects. They let you label and use that data later in your code.

Type of variables or keyword to define variables

  • var - Global scope and we can re-dclr and we can change value any time
  • let - Block Scope and we can't re-dclr. Must be Declared before use
  • const - Block Scope level we can't re-dclr and also we can't change its value except array or object
